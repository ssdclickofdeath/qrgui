About

Qrgui is an easy to use, minimalistic, GUI-based wrapper for qrencode, a QR code generator.

What does that mean? You can generate QR codes through a minimalistic, simple interface without using the command line at all.

Dependencies

Both qrencode and zenity must be installed for this script to work. zenity is used to make the GUI dialog boxes, qrencode generates the QR code.

To install the required dependencies with the apt-get package manager, run this command: sudo apt-get install qrencode zenity
